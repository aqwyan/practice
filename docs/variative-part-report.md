## 2. Суть выполненного вклада

| Параметр | Значение |
|----------|----------|
| **Тип вклада** | Создание интерактивной демки (улучшение документации) |
| **Затронутая тема** | CSS-свойство `display` |
| **Технологии** | HTML, CSS, JavaScript |

### Что было сделано:

1. Изучена структура репозитория и файлы документации.
2. Выбрана статья про CSS-свойство `display`.
3. Создана интерактивная демка на HTML/CSS/JS, которая позволяет:
   - выбирать значение `display` из выпадающего списка;
   - мгновенно видеть, как меняется расположение блоков;
   - читать пояснения к каждому значению (`block`, `inline`, `inline-block`, `flex`, `grid`).
4. Сделан Fork репозитория.
5. Через веб-интерфейс GitHub внесены изменения в файл `css/display/index.md`.
6. Создан Pull Request с подробным описанием внесённых изменений.
